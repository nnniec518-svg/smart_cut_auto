# 模型下载指南

## 方法一：自动下载（推荐）

### Windows 用户
双击运行 `download_model.bat` 文件，脚本会自动使用国内镜像下载模型。

### Linux/Mac 用户
```bash
python download_model.py
```

## 方法二：手动下载

如果自动下载失败，可以手动下载模型：

### 1. 设置镜像源
```bash
export HF_ENDPOINT=https://hf-mirror.com
```

### 2. 使用 huggingface-cli 下载
```bash
# 安装 huggingface-hub（如果未安装）
pip install huggingface_hub

# 下载模型
huggingface-cli download BAAI/bge-large-zh-v1.5 --local-dir ./models/BAAI/bge-large-zh-v1.5
```

### 3. 使用 Python 下载
```python
import os
os.environ['HF_ENDPOINT'] = 'https://hf-mirror.com'

from sentence_transformers import SentenceTransformer
model = SentenceTransformer("BAAI/bge-large-zh-v1.5", cache_folder="./models/sentence_transformers")
```

## 方法三：离线下载

如果网络完全不通，可以：

1. 在有网络的机器上下载模型
2. 复制整个 `models/BAAI/bge-large-zh-v1.5` 目录到本机
3. 或者使用 U 盘传输

## 下载完成后的配置

下载完成后，更新 `config.yaml`：

```yaml
models:
  # 使用新的中文模型
  embedding_model: "BAAI/bge-large-zh-v1.5"

  # 降级模型
  fallback_embedding: "sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2"
```

## 验证安装

运行测试脚本验证模型是否正确安装：

```bash
python test_model.py
```

## 模型对比

| 模型 | 优势 | 劣势 | 大小 |
|------|------|------|------|
| BAAI/bge-large-zh-v1.5 | 中文理解强，准确度高 | 模型较大，加载较慢 | ~1.3GB |
| paraphrase-multilingual-MiniLM-L12-v2 | 模型小，加载快，支持多语言 | 中文理解稍弱 | ~470MB |

## 常见问题

### Q: 下载速度很慢怎么办？
A: 可以尝试：
1. 使用代理/VPN
2. 切换到其他镜像源（如 ModelScope）
3. 在网络状况好的时候下载

### Q: 下载中断了怎么办？
A: 重新运行下载脚本，sentence-transformers 支持断点续传。

### Q: 磁盘空间不足怎么办？
A: 可以：
1. 清理 `models` 目录下的旧模型
2. 继续使用较小的 paraphrase-multilingual-MiniLM-L12-v2 模型

## 网络问题排查

如果遇到网络问题，可以尝试：

```bash
# 测试镜像源是否可访问
curl -I https://hf-mirror.com

# 或使用 ping
ping hf-mirror.com
```

如果镜像源不可用，可以尝试其他源：
- ModelScope: https://modelscope.cn/
- HuggingFace 官方（需要翻墙）
