## 🔧 手动配置

如果自动脚本不工作，可以手动配置：

### 1. 下载模型

```python
import os
os.environ['HF_ENDPOINT'] = 'https://hf-mirror.com'

from sentence_transformers import SentenceTransformer
model = SentenceTransformer(
    "BAAI/bge-large-zh-v1.5",
    cache_folder="./models/sentence_transformers"
)
```

### 2. 更新配置

编辑 `config.yaml`:

```yaml
models:
  embedding_model: "BAAI/bge-large-zh-v1.5"
  fallback_embedding: "sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2"
```

### 3. 验证安装

```bash
python test_model.py
```

## ⚠️ 常见问题

### Q: 下载速度慢怎么办？
A: 可以尝试：
1. 使用代理/VPN
2. 切换到其他镜像源
3. 在网络状况好的时候下载

### Q: 下载中断了怎么办？
A: 重新运行下载脚本，sentence-transformers 支持断点续传。

### Q: 磁盘空间不足怎么办？
A: 可以继续使用较小的 paraphrase-multilingual-MiniLM-L12-v2 模型。

### Q: 模型加载失败怎么办？
A: 检查：
1. 模型文件是否完整下载
2. config.yaml 配置是否正确
3. 运行 test_model.py 查看详细错误信息
